到达底部

JS里面获取元素的方法有很多，我们先了解一种最简单常见的获ID：document.getElementById("id"); //注意大小写

JS的事件 :通过 鼠标、键盘、页面变化 等都会产生事件,比如：onclick ondbclick onmouseover onmouseout onmousemove onmousedown onmouseup onmousepress onkeyup onkeydown(键盘 按下去) onkeypress onresieze(页面大小改变) onload

function 名字(){ };比如：function hide(){ document.getElementById("list").style.display = "none"; };

函数是不会主动执行的，执行函数的方法有很多种… 直接调用 aaa();比如：function hide(){ document.getElementById("list").style.display = "none"; }; hide();
事件调用 元素.事件 = aaa; 比如：document.getElementById("li-a").onmouseover = show;//注意这里show没有圆括号，有圆括号表示随着浏览器的加载而执行，而没有圆括号是表示需要 触发事件才能执行！！！ 

				document.getElementById("li-a").onmouseover = show;
				document.getElementById("li-a").onmouseout = hide;
				function show(){
					document.getElementById("list").style.display = "block";
				};
				function hide(){
					document.getElementById("list").style.display = "none";
				};
				以上的代码等同于：
				var liaDom =  document.getElementById("li-a");
				var lDom = document.getElementById("list");
				liaDom.onmouseover = function(){//匿名函数
					lDom.style.display = "block";
				};
				liaDom.onmouseout = function(){
					lDom.style.display = "none";
				};

一般为了解决脚本语言放置head结束标签前，脚本语言要用window.onload = function(){ };（用来优先加载所有dom元素）来包裹住,注意有时也会在body结束标签用它来包裹住，为了解决 由于标签元素和脚本语言之间有很大的障碍尺寸(如：很大的图片)，而没能加载完dom元素(图片、文字、标签等等)，所以要加window.onload = function(){ }; 

				js对HTML元素属性的操作读、写过程注意事项：
				1. 驼峰命名(函数或变量由多个单词组成时，第一个单词首字母小写，剩余的单词首字母大写)如：alert(oBox.style.marginLeft);不能写成alert(oBox.style.margin-left);	
				2. class不能直接写，而是写成className	
				3. 操作的样式较多，一般用css代替		
				4. 相对路径地址、颜色值不能在if语句用来判断，地址只能判断绝对路径。document.write(oImg.src);//打印出来~	
				5. input的type可以改变但是不兼容IE6 7 8，
				元素的浮动属性是不兼容IE6 7 8（ie：styleFloat  非ie:cssFloat）
				采用其他的方式规避这些不兼容的效果。
